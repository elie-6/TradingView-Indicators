# TradingView Indicators

A collection of custom TradingView indicators created in Pine Script.  

## Current Features

- **Session High/Low Rays + Open Lines (UTC+2)**  
  - Asia, London, and New York sessions  
  - Vertical dashed lines at session opens  
  - Horizontal rays from the session high/low candles  
  - Handles holidays/weekends automatically  

## Installation / Usage

1. Copy the Pine Script code from the relevant `.pinescript` file.  
2. Open [TradingView Pine Editor](https://www.tradingview.com/pine-script-docs/en/v4/).
3. Create a new strategy.
4. Paste the code and click **Add to Chart**.
